# Documentation Module (Knowledge Base)

## 职责 (Responsibilities)
本模块是项目的核心知识沉淀区，涵盖了 Java 核心、云原生、分布式架构及面试实战等深度笔记。

- **2025云原生**: 包含 Stage 1-3 的阶段性学习计划、周笔记及 AWS Bootcamp 实录。
- **Interview**: 包含分布式架构落地方法论等深度技术专栏。
- **QBank**: 针对 Java、后端、云原生的分类面试题库。

## 技术栈 (Tech Stack)
- **格式**: Markdown (主要) / AsciiDoc (复杂文档)
- **构建**: 支持 GitBook / GitHub Pages 自动化发布

## 发布指南 (Publishing Guide)
