# Backend Interview Module

## 职责 (Responsibilities)
面试高频考点源码级示例，涵盖 JVM、并发、Spring、云原生等。

## 技术栈 (Tech Stack)
- Spring Boot 3.2
- Testcontainers (Integration Testing)
- Micrometer (Observability)

## 快速启动 (Quick Start)
```bash
mvn clean install -pl backend-interview
```

## 扩展指南 (Extension Guide)
在 `com.renda.backend` 下按知识点分类创建子包并编写示例代码与集成测试。
